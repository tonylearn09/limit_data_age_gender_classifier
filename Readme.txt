How to use
1. Install python2, scikit-learn, numpy, scipy, scikit-image, dlib, opencv

2. get the shape_predictor_68_face_landmarks.dat file from:
http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

3. Training
    - python face_lbp.py
    - python train.py 
	(generate a pkl file(the Model), and output validation score)

4. Testing
    - python readfile.py [path_to_model_generated by train.py] [path_to_data_dir]
    (we have include a trained model: SVM_8496_3.pkl)
